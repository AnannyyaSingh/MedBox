# 🩺 MedBox – Symptom-Based Medical Recommendation System

MedBox is a full-stack medical web application that helps users get basic medicine suggestions by entering their symptoms. It acts as a simple decision-support system for preliminary healthcare guidance.

---

## 🚀 Features

- 🔍 Symptom-based medicine suggestions
- 💡 Rule-based recommendation logic
- 🌐 Interactive and responsive UI
- ⚡ Fast backend processing with Flask
- 📱 Beginner-friendly healthcare assistance system

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Tools:** Git, GitHub, VS Code  

---

## 🧩 How It Works

1. User enters symptoms (e.g., fever, headache)
2. Backend processes input using predefined logic
3. System returns possible medicine suggestions
